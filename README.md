# WorkDiary_MVVM
WorkDiary MVVM
<br>개발기간 : 2021.06.15 ~ 2021.07.01
<h2>기술 스택</h2>
- Android Kotlin <br>
- Room database를 활용하여 데이터 관리 <br>
- Room + LiveData + ViewModel 을 활용한 MVVM 아키텍쳐 구현
- AsyncTask 로 비동기
